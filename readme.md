## Requirements

- macOS 10.13+
- Xcode 13+
- Swift 5.5+

## Install

#### Swift Package Manager

```swift
.package(url: "https://github.com/wulkano/Aperture", from: "1.0.0")
```

#### Carthage

```
github "wulkano/Aperture"
```

#### CocoaPods

```ruby
pod 'Aperture'
```
